### 문제 설명

<p>국민대학교에서는 수강 신청, 성적 조회 등 과목의 구분이 필요할 때 ’과목 코드’를 사용한다. 과목 코드는 10자리로 이루어져 있으며, 7번째 자리는 영어 알파벳 대문자 또는 숫자, 8번째 자리는 하이픈(<span style="color:#e74c3c;"><code>-</code></span>), 나머지 자리는 숫자로 이루어져 있다.</p>
