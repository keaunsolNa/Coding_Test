<p>레이무는 상하좌우 중 한 방향으로 기계를 작동할 수 있다. 기계에 존재하는 모든 동전은 기계가 작동한 방향으로 이동하며, 동전이 이동한 경로에는 새로운 동전이 생겨난다. 레이무는 이를 보고, 기계의 모든 칸을 동전으로 채우기로 결심했다.</p>
