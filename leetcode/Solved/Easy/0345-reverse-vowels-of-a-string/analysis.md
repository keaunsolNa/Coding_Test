## Code Review

리뷰 대상 코드는 문자열의 모음만을 뒤집는 문제를 해결하는 자바 코드입니다.

1. **시간 복잡도**: O(n) - 문자열을 두 번 순회합니다. 첫 번째 순회에서는 모음의 인덱스를 찾고, 두 번째 순회에서는 모음을 뒤집습니다. 여기서 n은 문자열의 길이입니다.

2. **공간 복잡도**: O(n) - 문자열의 길이에 따라 모음의 인덱스를 저장하는 리스트의 크기가 결정되며, 문자열을 배열로 변환하여 저장합니다.

3. **풀이 접근법**: 두 개의 포인터(두 개의 인덱스)를 사용하여 문자열의 모음을 뒤집습니다. 먼저 모음의 인덱스를 리스트에 저장하고, 리스트의 처음과 끝을 교환하며 중간으로 이동하는 방식입니다.

4. **잘된 점**: 코드는 문제를 명확하게 이해하고, 모음의 인덱스를 리스트에 저장하여 효율적으로 뒤집는 방식을 사용했습니다. 또한, StringBuilder를 사용하여 문자열을 효율적으로 생성했습니다.

5. **개선 사항**: 코드는 이미 효율적으로 작성되어 있으나, 두 포인터를 사용하여 문자열을 직접 뒤집는 방식으로 개선할 수 있습니다. 즉, 모음의 인덱스를 리스트에 저장할 필요 없이, 두 포인터를 사용하여 직접 문자열을 뒤집는 방식입니다. 예를 들어, 두 포인터를 문자열의 시작과 끝에 위치시키고, 모음이 아닌 문자를 만날 때마다 포인터를 이동시키는 방식입니다. 이 방식은 리스트를 사용하지 않으므로 공간 복잡도를 O(1)로 개선할 수 있습니다.
