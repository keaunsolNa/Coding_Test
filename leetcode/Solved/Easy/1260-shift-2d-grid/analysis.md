## Code Review

**코드 리뷰**

### 1. 시간 복잡도
시간 복잡도는 O(n*m)입니다. 여기서 n은 격자(grid)의 행 수, m은 열 수입니다. 이는 격자 전체를 두 번 순회하기 때문입니다. 첫 번째 순회에서는 1차원 배열로 변환하고, 두 번째 순회에서는 이동된 배열을 생성합니다.

### 2. 공간 복잡도
공간 복잡도는 O(n*m)입니다. 이는 격자 전체를 1차원 배열로 변환하여 저장하고, 이동된 배열을 추가로 저장하기 때문입니다.

### 3. 풀이 접근법
이 풀이에서는 격자를 1차원 배열로 변환하고, 배열의 요소를 지정된 수만큼 회전시킵니다. 이는 배열의 인덱스를 조작하여 회전을 구현합니다. 이 접근법은 간단하고 직관적이며, 문제의 요구 사항을 잘 만족합니다.

### 4. 잘된 점
- 코드는 읽기 쉽고, 변수 이름이 명확합니다.
- 격자와 1차원 배열 사이의 변환을 잘 구현했습니다.
- 회전 로직이 간단하고 이해하기 쉽습니다.

### 5. 개선 사항
- 불필요한 출력문(`System.out.print`, `System.out.println`)을 제거하여 코드를 깔끔하게 유지할 수 있습니다.
