## Code Review

코드 리뷰입니다.

2. **공간 복잡도**: O(m*n)입니다. 입력 격자와 동일한 크기의 배열을 생성하고, 이동한 배열을 생성하며, 결과를 저장할 리스트를 생성하기 때문입니다.
