코드 리뷰
================

### 1. 시간 복잡도

해당 코드의 시간 복잡도는 O(n^2)입니다. 이유는 while 반복문 내에 for 반복문이 포함되어 있기 때문입니다. while 반복문이 n번 실행될 때마다, for 반복문은 n까지 반복되므로 시간 복잡도가 O(n^2)가 됩니다.

### 2. 공간 복잡도

해당 코드의 공간 복잡도는 O(1)입니다. 코드에서 사용되는 변수는 모두 상수 공간을 차지하며, 동적 공간 할당이 없기 때문에 공간 복잡도는 O(1)입니다.

### 3. 풀이 접근법

해당 코드는 그리디 알고리즘을 활용하여 문제를 해결하고 있습니다. 즉, 각 단계에서 가능할 때마다 게임에 가장 좋은 선택을 하는 방식으로 진행합니다. 또한, 게임의 상태를 단순화하여 alice의 턴이 언제인지 체크하는 변수를 사용하여 게임의 결과를 판단합니다.
