# 233. Number of Digit One

**Difficulty:** Hard | **Tags:** Math, Dynamic Programming, Recursion

## Description

Given an integer `n`, count _the total number of digit _`1`_ appearing in all non-negative integers less than or equal to_ `n`.

 

 **Example 1:**

 ```
Input: n = 13
Output: 6
```

 **Example 2:**

 ```
Input: n = 0
Output: 0
```

 

 **Constraints:**

 - `0 <= n <= 109`
