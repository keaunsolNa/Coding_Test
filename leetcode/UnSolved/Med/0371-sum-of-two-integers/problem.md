# 371. Sum of Two Integers

**Difficulty:** Medium | **Tags:** Math, Bit Manipulation

## Description

Given two integers `a` and `b`, return _the sum of the two integers without using the operators_ `+` _and_ `-`.

 

 **Example 1:**

 ```
Input: a = 1, b = 2
Output: 3
```

**Example 2:**

 ```
Input: a = 2, b = 3
Output: 5
```

 

 **Constraints:**

 - `-1000 <= a, b <= 1000`
